# 使用说明

## 🎮 游戏启动

### 快速启动（推荐）
1. 双击 `index.html` 文件
2. 游戏将在默认浏览器中打开
3. 点击"开始游戏"按钮
4. 开始游玩！

### 使用本地服务器
1. 双击 `start-server.bat` 文件
2. 打开浏览器访问 `http://localhost:8000`
3. 开始游戏！

## 🎯 基本操作

### PC 操作
- **移动鼠标**：控制小球释放位置
- **点击鼠标**：释放小球

### 手机操作
- **触摸屏幕**：选择释放位置
- **松开手指**：释放小球

## 🎨 自定义素材

### 上传图片
1. 点击右上角 ⚙️ 按钮
2. 点击"批量上传"按钮
3. 选择最多 11 张图片
4. 图片按顺序分配到 1~11 级

### 图片要求
- **格式**：JPG、PNG、GIF、WebP
- **尺寸**：建议 200x200 像素
- **形状**：系统会自动裁剪为圆形

### 恢复默认
1. 点击右上角 ⚙️ 按钮
2. 点击"恢复默认"按钮
3. 确认后恢复为系统默认素材

## 📖 图鉴系统

### 查看图鉴
1. 点击右上角 📖 按钮
2. 查看所有 11 个等级的素材
3. 已解锁的等级显示为彩色
4. 未解锁的等级显示为灰色

### 解锁条件
- 合成到对应等级即可解锁
- 解锁状态保存在本地
- 重新开始游戏不会丢失解锁记录

## 💡 游戏技巧

### 合成技巧
1. **控制位置**：尽量将小球放在容易合成的位置
2. **规划路线**：提前规划好合成路线
3. **避免堆积**：不要让小球堆积得太高
4. **保持空间**：保持游戏区域有足够空间

### 得分技巧
1. **快速合成**：快速合成可以获得更高分数
2. **高等级优先**：优先合成高等级小球
3. **连续合成**：连续合成可以获得连击奖励

## 🛠️ 故障排除

### 游戏无法启动
- 检查是否使用现代浏览器
- 检查网络连接
- 尝试刷新页面

### 图片上传失败
- 检查图片格式是否支持
- 减小图片尺寸
- 清除浏览器缓存

### 游戏卡顿
- 减小上传图片的尺寸
- 重新开始游戏
- 关闭其他标签页

### 数据丢失
- 避免清除浏览器数据
- 不要使用无痕模式
- 定期备份自定义素材

## 📊 游戏数据

### 本地存储
- **素材数据**：Base64 格式存储在 localStorage
- **解锁状态**：JSON 格式存储在 localStorage
- **最高分**：数字格式存储在 localStorage

### 存储限制
- localStorage 通常限制为 5-10MB
- 每张图片约占用 20-50KB
- 11 张图片总计约 200-500KB

## 🎵 音效说明

### 合成音效
- 合成时播放简短的音效
- 音调随等级升高而升高
- 使用 Web Audio API 生成

### 游戏结束音效
- 游戏结束时播放低沉的音效
- 提示玩家游戏结束

### PC 端

## 🔄 游戏流程

1. **开始游戏**
   - 点击"开始游戏"按钮
   - 进入游戏主界面

2. **释放小球**
   - 移动鼠标/触摸屏幕选择位置
   - 点击/松开释放小球

3. **合成升级**
   - 相同等级小球碰撞合成
   - 获得相应分数

4. **游戏结束**
   - 小球堆积超过虚线
   - 显示得分和最高分

5. **重新开始**
   - 点击"再来一局"按钮
   - 开始新的游戏
